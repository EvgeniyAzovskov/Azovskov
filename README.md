### 👨‍💻 Обо мне:

Бывший инженер-технолог и мастер на производстве, имеющий опыт работы со сложными
техническими системами, стандартами качества (ISO, ТУ) и управлением процессами. За
время работы на производстве научился выявлять уязвимости в технологических процессах,
систематизировать отклонения и доводить продукт до соответствия строгим требованиям.
Прошел полную профессиональную переподготовку на тестировщика ПО. Понимаю SDLC,
виды тестирования, умею писать тест-кейсы, чек-листы, заводить баг-репорты, работаю с
баг-трекинговыми системами (например, Jira/YouTrack). Знаком с основами API тестирования
(Postman) и SQL.
Моя «производственная» внимательность к деталям, системное мышление и ответственность за
качество конечного продукта стали основой для перехода в IT. Ищу позицию Junior QA Engineer,
где смогу применять свой инженерный склад ума и дисциплину для повышения качества
продукта
